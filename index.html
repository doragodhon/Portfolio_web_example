Who Am I ?


    สวัสดีครับ ผมนายณัทพงศ์ อักษราลิขิตสันติ สถาบันเทคโนโลยีพระจอมเกล้าเจ้าคุณทหารลาดกระบัง กำลังศึกษาอยู่คณะเทคโนโลยีสารสนเทศ แขนงมัลติมีเดียสื่อประสมและเกมส์ นักศึกษาชั้นปีที่ 3 ผมเป็นคนที่เข้าสังคมได้ดี ปรับตัวได้ง่าย สำหรับการทำงานเป็นทีมแล้ว จากประสบการณ์ที่ผมได้ทำโปรเจคกลุ่มนั้น ผมเป็นคนที่คอยติดตามงานอยู่เสมอ สอบถามความคืบหน้า และช่วยเหลือคนในกลุ่มเมื่อเกิดปัญหาขึ้น ผมมีความสามารถในการใช้งานโปรแกรมต่าง ๆ เช่น Maya Zbrush ในการสร้างตัวละคร, การใช้งาน Unreal Engine 4 ในการพัฒนา สร้างเกม Landscape สร้างแสงสีให้กับฉากได้ออกมามีชีวิตชีวามากขึ้น อีกทั้งผมยังสามารถใช้ ภาษาโปรแกรมมิ่งอย่างเช่น Javascript HTML CSS



What I'm interested ?



    ผมมีความสนใจในการสร้างเกม เนื่องจากผมมีความคุ้นชินกับคอมพิวเตอร์มาตั้งแต่สมัยเด็ก ผมหลงใหล สเน่ห์ของมัน ความสร้างสรรค์อันไม่มีที่สิ้นสุด ผมได้ลองเล่นเกมมาหลากหลายรูปแบบ ทั้งมุมมองบุคคลที่ 1 และ 3 เกมจำลองชีวิตจริง ฯลฯ เกมแต่ละประเภทนั้นก็มีสเน่ห์เป็นของตัวมันเอง มีความโดดเด่น เอกลักษณ์ และรูปแบบการเล่นที่แตกต่างกัน ผมจึงเริ่มมีความคิดอยากทำ เกมเป็นของตัวเองบ้าง ผมจึงได้ตัดสินใจเลือกเรียนสาขาที่มีการสอนสร้างสิ่งของต่าง ๆ เพื่อนำไปใช้ในเกม ทั้งเรียนรู้ในจากวิชาและ Self-learning

ผมมีความสนใจเป็นพิเศษกับงานที่เกี่ยวข้องกับเอฟเฟกต์หรือ VFX การใส่ VFX ทำให้ฉากมีความตื่นตา มีสีสัน อีกทั้งยังช่วยแก้ปัญหาเมื่อในการ ถ่ายทำฉากนั้นไม่สามารถทำได้จริง เป็นไปไม่ได้หรือเป็นไปได้ยาก ผมให้ความหมายของ VFX เสมือนศิลปะ เป็นการแต่งเติมสิ่งที่ไม่มีอยู่จริง ให้กับฉากหรือวีดีโอ เติมแต่งความสร้างสรรค์อันไม่มีที่สิ้นสุดลงไป



My Skills

     Unreal Engine เป็นเกมเอนจินในการสร้างวิดีโอเกมที่พัฒนาขึ้นโดยบริษัท Epic Games โดยเอนจินนี้จะเป็นแบบ Open Source ดังนั้นผู้ใช้หรือบริษัทแต่ละรายสามารถปรับปรุงและปรับให้เข้ากับความต้องการได้ ใช้เพื่อสร้างเกมส์หลากหลายประเภท ทั้งมุมมองบุคคลที่ 1 และ 3 หรือ 2DSidescroll นอกจากนี้ยังโดดเด่นเรื่องของ ระบบแสง การสร้างเนื้อหาวัสดุจำนวนมาก

    Maya เป็นโปรแกรมสำหรับสร้างโมเดล 3 มิติ และอนิเมชั่น 3 มิติ โดยรองรับมาตรฐานต่างๆ ด้านงานกราฟิก 3 มิติทุกประเภท เช่น 3D Visual Effects, Computer Graphics และเครื่องมือในการสร้าง การ์ตูน Animation สามารถสร้างผลงานทีวี, พัฒนาเกม และงานออกแบบต่างๆได้อย่างรวดเร็ว

     ZBrush เป็น Software สำหรับการปั้นโมเดลที่มีรายละเอียดสูง เริ่มพัฒนาจาก 2.5 D ที่ค่อนข้างยากในการใช้งานจนปัจจุบันได้เพิ่ม ฟังค์ชั่นในการปั้นโมเดลและใส่เทคเจอร์ที่มีรายละเอียดมากๆได้ อีกทั้งยังมีความสามารถในการ Export ข้อมูล Mesh, Texture และข้อมูล Normal maps เป็นต้น ออกไปใช้ร่วมกับโมเดลที่เป็น Low Poly ในโปรแกรม 3D อื่นๆ เพื่อใช้ทำแอนิเมชั่น ต่อไปได้
